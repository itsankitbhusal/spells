# Spells

A spellbook application built with React, TypeScript, and Vite.

## Features

- Search Spells
- Add Spells to Favorites
- View Spell Details

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- React Query
- Axios

## Setup

```bash
nvm use # use node version v20.9.0
pnpm i
pnpm run dev
```
